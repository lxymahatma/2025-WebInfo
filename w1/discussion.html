1. Difference between Website and Web Application:

  • Website: Mainly informational. Users view or read content (e.g., news sites). Usually static or updated occasionally.
  • Web Application: Interactive. Users input data and get dynamic responses (e.g., Gmail, online banking). Frequently updated and user-driven.

2. What Makes a "Good" Web Application:

  • Fast loading speed
  • Easy navigation
  • User-friendly and intuitive interface
  • High security and reliability
  • Accessible to users with disabilities
  • Mobile-optimized experience
